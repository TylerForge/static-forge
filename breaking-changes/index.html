Breaking changes

Legacy TUX Guidelines -> Forge​

While Forge builds on legacy TUX guidelines, we’ve updated design guidance to improvement components for usability concerns or for Tyler’s domain specific needs. We’ll be tracking those changes here for transparency.

UI region
Legacy guidance
Forge guidance
Reason for change
Navigation links
Input style Underline with floating label Outlined inputs Usability issues: Outlined inputs better recognized as traditional inputs.
Input height 56px Default (Desktop): 40px; Roomy (Mobile): 56px Usability issues: 56px inputs are too large for input-heavy forms on desktop; pushes data out of the viewport
Icon resting color Black, 87% Black, 54%
Floating label Floating label with animation Remove animation; persist floating label Usability issues: animation is distracting in input-heavy forms Technical issues: occasionally label width isn’t available when the component renders, labels can animate randomly
Datatable editing Dialog edit (Material) Coming soon
Controls Use Amber 500 Use Indigo a200 ADA issues: Amber doesn’t provide sufficient color contrast for our contrast; Opted for Indigo a200 to provide sufficient contrast against white or Grey-50 background.
Button ripple Ripple animation No ripple animation Technical issues: Technical overhead isn’t worth the value-add. (alignment issues, extra listeners added to window)
Button text ALL CAPS Sentence case Usability issues: ALL CAPS can cause issues with built in language translators ADA issues: ALL CAPS styling can cause issues with screenreader performance
Outlined button Use light grey outline in resting state Use indigo outline ADA issues: An indigo border provides sufficient color contrast for ADA.
App background n/a Gray 50 Design: Provide consistent app background standard; provides additional contrast between white cards and background; elevates visual emphasis on content itself
Filter facets Display facet categories on separate raised cards Display facet categories on a single card with dividers Design: Reduce visual color in order to provide dominant visual emphasis on filter text
