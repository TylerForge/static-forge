Guidance

The omnibar is used for both back office applications and public-facing web sites.

Best Practices

The branding logo and application title typically navigate the user back to the start screen or home page of the application or web site.

The optional menu launches the navigation drawer with the main content or functional areas of an application.

Tenant Branding for Omnibar

Apps that are public facing can be branded to reflect a municipality or organization identity.

Do.

Caution.

Place municipality name first, app name second. 

Between the two: there should be 20px of spacing, a rule “|”, and another 20px of spacing. 

We need to be very conscious of the names we are giving our applications, and make sure that they are meaningful. “City of Renton Portico” for example, means much less to a citizen than “City of Renton Utility Billing.”

Do.

Do

Page title and/or Switcher at the top of the left-hand navigation.

If your omnibar has a search, and you don’t have room for the Municipality Name | App Name, then minimize the search bar and show just the Material Search icon (magnifying glass).

Implementation

The omnibar component is used as the application header, and should stay visible and positioned at the top of the application at all times. This component contains the Tyler branding logo, the title text, optional menu button, as well as customizable slots for search, actions, and profile.

Usage

<tcw-omnibar>
  <tcw-omnibar-search-input slot="center">
    <button class="material-icons mdc-top-app-bar__action-item" slot="action">cakebutton>
  tcw-omnibar-search-input>
  <button class="material-icons mdc-top-app-bar__action-item" slot="end">appsbutton>
  <tcw-circle-avatar text="First Last" slot="end">tcw-circle-avatar>
tcw-omnibar>

Slots

NameDefaultAttribute
centerPlaces content in the center of the omnibar after the title text.


endPlaces content aligned to the right side of the omnibar.

Attributes

Name Default Attribute
title-text '' The text that displays in the omnibar title.
menu true Controls the visibility of the menu icon.
type Controls the inner top-app-bar component type.
scroll-target window The element to attach the scroll listener.
dense true Controls the dense state.
raised false Controls the visibility of the static elevation shadow.
