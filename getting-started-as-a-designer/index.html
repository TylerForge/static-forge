Getting started: As a designer

 Grab our Forge Sketch symbols, the default Material sketch library, and our recommended plugins.

face Heads up, this page is still in progress! Check back soon to see the final product 🙂

Learn about Forge

  • Read the Forge design guidance to familiarize yourself with our foundational styles, patterns, and components.
  • Use Sketch (if you’re a designer on a mac) or Figma (if you’re a product owner or designer using a PC) to create designs with Forge components. 

Getting started with Sketch

If you’re on a mac, download Sketch here. You’ll need a license to use the software – contact kathy.wolf@tylertech.com for more information. 

Sketch has some great beginner guides for getting started here.

Required fonts

Roboto is our official Tyler Forge font. Download it here. Be sure to install Thin, Thin Italic, Regular, Regular Italic, Medium, and Medium Italic.

Required plugins

For designers using Sketch, we use Invision for prototyping, sharing, commenting, and developer handoff. To use Invision with Sketch, download the Craft plugins here. 

Once you’ve installed Craft, you’ll see it in your command bar in mac. You’ll need to open it and create an account before you can view the Craft command bar in sketch. 

Open the craft plugin from your mac command bar.

Then sign in to craft before you can see it in Sketch.

Once you’ve signed in to Craft, open Sketch to see the suite of Craft options.

Craft includes eight options: Sync, Prototype, Preview, DSM, Data, Duplicate, Freehand, and Stock.

Learn more about Craft here. 

Recommended plugins

Download Sketch Runner first. (Runner allows you to install new apps and execute commands within Sketch).You can use the free version.

Then you can use Sketch runner to add:

  • Stark (Design with accessibility in mind)

Adding the Forge library

To use Forge components in Sketch, use the Forge design library. We are in the process of updating our symbols to use nested layer styles, so we actually have two libraries at the moment (in the future we should just have one).

Download the Forge Theme 0.1.0 file. Then, add it to your libraries in Sketch.

Navigate to Sketch > Preferences > Libraries.

Then, hit “Add library” and select the Tyler Forge Theme 0.1.0 file from your downloads. 

Now, you’ll be able to add symbols from the Forge library when you click “Insert” in Sketch.

 

Adding the Forge icons

To use Forge icons within Sketch, you’ll need to add the Forge Icon File as a library as well. Once you’ve added it as a library, you can access layer styles to color icons correctly.

Adding the DSM library

Next, add the DSM library in Sketch. Any new components will be added to the DSM library instead of the Forge Theme 0.1.0 doc. 

Our DSM sketch library is still in beta.  We have a limited number of seats for our DSM plugin at the moment. Reach out to emily.roller@tylertech.com for access.

Learn how to add elements from the DSM library into your document here.

If you have access, hit the DSM icon in your craft plugin. Ensure that “Forge Test Library 2” is selected. 

Using layer styles

 The DSM symbols use layer styles. To learn more about layer styles in Sketch, check out this article. 

In order to use our Forge layer styles in your document, you’ll need to add our typography and color style artboards. 

Navigate to the “Styles” folder in DSM, and drag all the available artboards into your document. Once you’ve done that, you can delete them; our layer styles will have been saved locally. 

 

Once you’ve imported the layer styles, you’ll be able to set typography, color, and elevation from our Tyler Forge styles within your symbols. 

Now you’re all set to use our Forge components in Sketch! Have questions or see a component missing? Let us know!

Pulling in updates

The DSM library will updated as we add additional components and modify existing ones.

With DSM you can quickly sync library changes straight to your Sketch file. To update symbols (components in DSM), text styles, and layer styles, open the DSM tool and click the sync icon.

In order for components to update, they must first be defined as symbols in Sketch. To push changes for non-symbol components or other elements, drag and drop each item from your DSM library to your file.

Learn more about Version Control in DSM here.

FAQs

Should I update Sketch when it tells me to?

Yes! Staying up to date with Sketch will help you take advantage of new Sketch features. 

What happens if the library is updated?

When the DSM library is updated, you’ll be able to pull updates into your document. Ensure that you’re using symbols and layer styles instead of detaching components from components. That way, if there are updates made to specific components, you’ll be able to pull them in automatically. 

Can I break the Forge library?

You cannot break or change the library when viewing the library in your project files. Only users with specific permissions can make changes in the library.
