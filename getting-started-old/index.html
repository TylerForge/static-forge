Tyler Forge

Tyler Forge is a design system that uses shared, reusable components to empower teams to efficiently create applications that express Tyler’s core values.
Components

Individual pieces of an interface that combine together to display data and add value. Examples: cards, data tables, alerts

Patterns

Methods of designing interactions that combine components in a meaningful way. Examples: editing, bulk adding, saving, onboarding

Layout

Groups of patterns and components that create your app structure. Examples: laying out search pages or faceted results, detail pages
