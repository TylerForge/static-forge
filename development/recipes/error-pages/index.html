Error pages

Recipes for the page state component

Guidance

Our page state errors use illustrations hosted in our CDN (content delivery network). To learn more about the CDN, check out our CDN guide.

The CDN provides a static URL to host our illustrations. It facilitates better performance, faster response time, and provides a non-changing URL to use in your applications. 

You’ll see that the recipes below use links to the CDN for our illustrations instead of hosting them with the project itself. You can copy the HTML as is provided below to render our page states – without having to host the illustrations yourself. 

Hit Run Project on the following examples and choose either Editor or Both to view the HTML.
