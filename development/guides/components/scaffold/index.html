Scaffold

With a few lines of code, have the framework of your application up and running with responsiveness, spacing, and typography taken care of.

Overview

The scaffold component is intended to be used as a way to structure an application using a common layout. The layout takes up the entire viewport height and width, while allowing for dynamically specifying content in each slot. The component gracefully supports any combination of slotted content.

This component uses CSS grid layout, and is supported by all browsers (including IE 11).

Example code

<tcw-avatar text="First Last">tcw-avatar>

The snippet above is the exact code used as the foundation of the Forge website.

Slots

Name Description
header This slot places content at the top of the viewport.
body This slot places content below the header slot and expands to the bottom of the viewport, or to the footer if specified.
body-left This slot places content to the left of the body.
body-right This slot places content to the right of the body.
body-footer This slot places content at the bottom of the body.
footer This slot places content below the body at the bottom of the viewport.
