Prerequisites

The following guide will ensure that your system is ready to install and develop web components using the TCW tools.

Node.js

Install the latest version of Node.js (minimum required version for using the TCW CLI is 8.3.0). This will also install npm (Node Package Manager), which will allow you to install the TCW packages.

Web Components

You should be up-to-speed with the latest information on web components, and the terms associated with them.

Please refer to the following links for more information:

  • Web Components
  • Custom Elements
  • Shadow DOM
  • HTML Template Element
  • Building Components
  • Web Components Polyfills

Material Design Components

The TCW web component library relies heavily on the Material Components Web library. This library is built and maintained by Google. It provides vanilla JavaScript components, and CSS styles that are up-to-date with the latest material guidelines and specs.

The TCW library uses a combination of the Sass utilities and JavaScript components to create custom elements that are easy to consume within a web application.

It’s important that you are familiar with the components in this library, as well as the CSS classes that are available to you. Using their documentation and examples will help you better understand and consume the TCW web components.

Technologies

Below is a list of technologies that are used across the TCW toolset and libraries. It’s worth familiarizing yourself with these as you’ll be using them in one form or another.

  • TypeScript
  • Sass
  • Web Components
    • Custom Elements
    • Shadow DOM
    • HTML Template Element
  • ES Modules
  • TSLint
  • StyleLint
  • Node.js
  • npm
