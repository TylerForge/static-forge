Theming

Theming is achieved via CSS variables for components that render in the shadow DOM.

Guidance

Theming is achieved via CSS variables for components that render in the shadow DOM. Each component is able to provide style hooks that allow its look and feel to be modified if desired. These style hooks are documented for each component, and only expose certain styles to be modified.

If you would like to be able to modify a style that doesn’t provide a hook, please contact us!

Alternatively, some components render in the light DOM and are subject to standard CSS overrides.
