Introduction

The following guides offer information about the Tyler Components Web library, and how to use it.

Each component provides in-depth documentation about how to use it, how to style it, and what its API looks like. Please refer to the individual component documentation for details.

Some elements on the page don’t require a custom element, such as inputs and buttons for example. For those we just provide CSS styles that can be applied to give them the Tyler look and feel. This can be achieved easily by adding a single class to the elements. For these types of elements, please refer to the individual style documentation under the “Styles” section.
