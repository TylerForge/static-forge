MapView

MapView is a framework agnostic, JavaScript map component built on Esri’s ArcGIS JavaScript API (version 3.x).  It provides a range of functionality, from basic map viewing to a feature-rich integration with a variety of ArcGIS services. 

Interactive demo

Guidance

Spatial location is a common element for public sector organizations, facilitating views between data, information and end-users.  With the use of a Geographic Information System (GIS), this relationship is easily seen and understood.  To further define this relationship, mapping provides a powerful tool to aggregate many sources of data and visualize how they affect each other.  A dataset that contains a spatial location through coordinates (latitude and longitude) or an address can be visualized using MapView.  User GIS data and layers can be added through the StartupConfig construction parameter.  Records in these datasets can be selected in the map to be used in the host program and vice versa.  The MapView component should be used to drive the application through the visualization and selection of these spatial datasets.

Functionality

MapView comes packed with functionality that consuming applications can use out-of-the-box.  This functionality includes:

  • Displaying one of the 20+ default Esri basemaps, or custom basemaps provided by the client.
  • Displaying and interacting with client’s GIS data through custom Esri Map and Feature Services.
  • Visualizing application-specific information by adding and interacting with point graphics. (polygon and polyline graphics coming soon!)
  • Customizing graphic styling and providing custom application-specific actions to graphics.
  • Clustering of graphics added to the map by the application.
  • Querying a client’s GIS service(s) for spatially selecting data.
  • Basic sketching functionality. (complete sketch and graphic edit functionality coming soon!)

Client-Facing Terminology

MapView is a map control that can be embedded in any web application.  A dataset that contains a spatial location through coordinates (latitude and longitude) or an address can be visualized using MapView.  Records in these datasets can be selected in the map to be used in the host program and vice versa.  A number of tools are included in MapView and the consuming product has control over the tools available in the map.  User GIS data and layers can be added to MapView through the application’s GIS Configuration page or through a config file included with the control.

MapView utilizes two modes to accommodate the varied needs of consuming applications, an embedded mode and panel mode.  When the map is in embedded mode the following tools are surfaced for users.

Address Search

The Address Search bar zooms and pans the map to an address provided by the user.  The tool utilizes ESRI’s world geocoding service unless a different geocoding service is provided in the configuration.  Suggest functionality is also provided as a user types and address.   List of suggested addresses will be provided allowing the user to click on a standard address provided by the geocoding service.

Navigation Tools

Tools are provided to change the extent of the map.  The Zoom to Current Items tool will change the extent of the map to the current features in the map.  Use the Zoom to Full Extent to change the extent of the map to the full extent of your map service.  The Find my location tool will pan the map to the coordinates provided by the location services of your device.  Also included are zoom in and zoom out tools to zoom the map in or out based on a set increment.

Info Window

When a user clicks on a feature in the map an info window will point to the feature and provide attributes as defined by the configuration.

Feature Clustering

If a dataset contains a large number of features clustering can make it easier to visualize and understand your data.  Be default the map will group point features that are within a specific distance of one another into one symbol.  The number of features represented by the cluster will be labeled on top of the symbol.  Clicking on the symbol will change to extent of the map to the features included in that cluster.  Double click anywhere on the map to turn off feature clustering.

When the map is in panel mode additional tools (see below) are surfaced for users.

Attribute Search

The attribute search provides user the ability to select assets or GIS features through an attribute search of any GIS attribute in the layers provided by the services added to the map.

Basemap

The basemap picker provides a list of the basemaps available for the map.  The standard ESRI basemaps or user defined basemaps can be made available in the basemap picker.

Bookmarks

The Bookmark tool gives users the ability to save and name a specific spatial extent.  Users can open the bookmark tool at any time and change the extent of the map to the bookmark of their choice.

Print

With the print tool users can export an image of the current extent providing a map title and using a map layout of their choice.

Select

Use the select tool to graphically select features in the map.  There are three options for graphic selections.  A polygon selection tool, a free hand selection tool and a buffer tool.  The buffer tool will draw a circle of the radius the user provides and select all the features within that area.  The layer list in the selection tool defines the layers that will participate in the feature selection.

Layers

The Layers tool provide a list of the layers available in the map and the ability to turn layers visibility on or off.  A transparency slider is also available to adjust the transparency of a layer.

Legend

View the legend for the currently visible layers in the map.

Draw / Measure

Use the draw/ measure tool to draw graphics on the map (Text, Points, Lines, Polygons).  The measure tool allows user to measure lengths, areas and points.

Clear

The clear tool clears all graphics and selections that are currently on the map.
