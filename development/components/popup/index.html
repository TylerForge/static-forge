Popup

Use the popup component to display a floating element that gets (smartly) positioned next to an arbitrary target element. The popup can contain any content, and is also used as the core component behind many other TCW components such as the datepicker, select, menu… etc. It is configurable and abstracts away the need to worry about positioning and scrolling when creating floating elements.

Installation

This component can be installed as an individual npm package via @tyler-components-web/popup

To install the full TCW component library package, see the getting started guide.

Usage

Popups typically aren’t created in the DOM directly, but they can be created dynamically using JavaScript:

const popup = document.createElement('tcw-popup');
popup.targetElement = document.getElementById('some-button');
popup.textContent = 'Popup content';
popup.open = true;

Angular usage

The @tylertech/tyler-components-web-angular adapter library offers an easy-to-use Angular directive for showing popups.

See the following example that shows how to use a popup all through Angular’s template syntax using ng-template:

<tcw-button type="raised">
  
  <button #popup="tcwPopup" [tcwPopup]="popupTemplate" (click)="popup.open()" tcwPopupCloseOnBlur="true">Show popupbutton>
tcw-button>


<ng-template #popupTemplate>
  <div>
    <div>Popup Contentdiv>
    <tcw-button>
      <button (click)="popup.close()">Closebutton>
    tcw-button>
  div>
ng-template>

The directive is responsible for dynamically creating an embedded view to place in the popup, while preserving the binding context of the location where the template is defined.
