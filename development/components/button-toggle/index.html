Button Toggle

Toggle buttons can be used to group related options. To emphasize groups of related toggle buttons, a group should share a common container.

Installation

This component can be installed as an individual npm package via @tyler-components-web/button-toggle

To install the full TCW component library package, see the getting started guide.

Button toggle group properties/attributes

Property

Attribute

Description

value

value

Gets/sets the selected value.

multiple

multiple

Controls whether multiple buttons can be selected at the same time. False by default.

stretch

stretch

Controls whether the child button-toggle elements will stretch to fill the available space.

mandatory

mandatory

Gets/sets whether a selection must be made at all times. False by default.

vertical

vertical

Controls whether the layout is vertical. False by default.

disabled

disabled

Gets/sets the disabled state of the component as a whole.

dense

dense

Applies the dense style to each child button-toggle component.

options

n/a

Allows for button-toggle components to be rendered via code instead of HTML.
