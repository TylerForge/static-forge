Omnibar notification button

The omnibar notification button component is a familiar place for users to access their system notifications. Use it within the omnibar to provide a consistent experience across Tyler applications.

Installation

This component can be installed as an individual npm package via @tyler-components-web/omni

To install the full TCW component library package (recommended), see the getting started guide.

Guidance

Use this component to allow for the user to access their notification information from a familiar location in the omnibar. This component provides the correct icon to use, and has facilities for displaying an indicator badge to let the user know they have notifications.

Usage

The notification button currently just provides a consistent icon button, and facilities to show the notification count (indicator). Listen for the click event on the element to show your notifications if necessary.

It is expected that this component be placed in the action slot of the omnibar.

Example
<tcw-omnibar title-text="Title text">
  <tcw-omnibar-notification-button slot="action">tcw-omnibar-notification-button>
tcw-omnibar>

Note: The goal in the future is to provide a common notification popup with this component for enhanced functionality, but the infrastructure to support this across Tyler does not exist yet… Stay tuned.

Properties/attributes

Property

Attribute

Description

count

count

Gets/sets the count text to display in the badge.

dot

dot

Controls whether the badge is displayed as a dot, or displays the count text (default).

showBadge

show-badge

Controls the visibility of the badge indicator.

theme

Sets the theme to use for the badge indicator. See badge component docs for valid theme values.
