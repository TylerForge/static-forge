Product Icon

The product icon component is used to generate the long shadow effect on a material icon.

Installation

This component can be installed as an individual npm package via @tyler-components-web/product-icon

To install the full TCW component library package, see the getting started guide.

Usage

The tcw-product-icon component facilitates the creation of the long shadow effect used primarily on application icons in the app drawer. It takes in a color from the material color palette and automatically adjusts the color of the shadow based on the color being used.

Properties/Attributes

Property

Attribute

Description

color

color

The color of the circle container that holds the icon. Accepts colors from the material palette (ie. 'indigo-500', 'deepPurple-500').

size

size

Sets the size of the circular container (defaults to 64px)

shadow

shadow

Boolean value, enables or disables the long shadow effect.

iterations

iterations

This is used to generate the length of the shadow. Set this to exactly half of the size attribute to achieve the correct effect per UX.

mode

mode

Accepts 'icon' & 'text'. When using icon mode, make sure to pass in the name of the material icon you want to use. When using text mode, keep the character count to just 2.

Slots

Default

The default slot will place content within the circle container
