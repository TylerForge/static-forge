Omnibar profile

The omnibar profile is a common location for a users’ personal information, as well as a means for navigating to their profile or signing out.

Installation

This component can be installed as an individual npm package via @tyler-components-web/omni

To install the full TCW component library package (recommended), see the getting started guide.

Guidance

Profile information should always be displayed in the profile slot (far right hand side) of the omnibar. This is a combination of a user avatar, profile card (within a popup), and a sign in/out button. All of these components can be used together to achieve the proper design, but for convenience, TCW offers the tcw-omnibar-profile-button component. This component can be used to easily ensure that the proper components and styles are used.

Usage

As noted above, the tcw-ominbar-profile-button component includes all of the necessary components that are used to compose the profile information for a user. It is intended that you use this component for consistency, but it’s worth noting that the components are all separated out for instances where this “convenience” component cannot be used.

Example
<tcw-omnibar title-text="Title text">
  <tcw-omnibar-profile-button slot="profile" avatar-text="First Last" full-name="First Last" email="first.last@tylertech.com">tcw-omnibar-profile-button>
tcw-omnibar>

Note: The profile card does NOT contain the sign in button. It is expected that your application will handle toggling between whether the sign in button is visible or if the profile-button is visible based on the current sign in status of the user.

Properties/attributes

Property

Attribute

Description

avatarImageUrl

avatar-image-url

Gets/sets the URL to load for the users' avatar image.

avatarLetterCount

avatar-letter-count

Gets/sets the number of characters to show in the avatar when an image URL is not used.

avatarText

avatar-text

Gets/sets the text to use for the avatar.

fullName

full-name

Gets/sets the name for the user to display in the profile card.

email

email

Gets/sets the email for the user to display in the profile card.

signOutButton

sign-out-button

Gets/sets whether the sign out button is visible in the profile card or not.

profileButton

profile-button

Gets/sets whether the profile button is visible in the profile card or not.

Events

Name

Description

tcw-omnibar-profile-card-sign-out

Emits when the "Sign out" button is clicked in the profile card.

tcw-omnibar-profile-card-profile

Emits when the "Profile" button is clicked in the profile card.
