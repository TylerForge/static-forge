Label Value

This component is simply a structural component that ensures consistent height, spacing, and typography of the label and value that is displayed.

Installation

This component can be installed as an individual npm package via @tyler-components-web/label-value

To install the full TCW component library package, see the getting started guide.

Guidance

Use the label-value component when you have data that needs to be displayed, but cannot be edited. This is generally ideal for a read-only screen, or toggling between viewing and editing data values.

This component is simple, but it provides consistency and ease of use through slotted content to remove the need for developers to worry about CSS styles.

Slots

Name

Description

label

Represents the label and has the proper typography applied.

value

Represents the value and has the proper typography applied.

icon

Places top-aligned content before the label and value slots.

Properties/Attributes

Property

Attribute

Description

empty

empty

Controls the typography of the value text when a no value can be displayed.

density

density

Intended to be used to match the density height of the text-field and select components when used alongside each other in a form. Valid values: default, roomy, dense

align

align

Controls the cross-axis alignment of the text. Valid values: left (default), center, right.
