Divider

This component is useful for separating sections, list items, or any other elements that are intended to be visually grouped separately.

Installation

This component can be installed as an individual npm package via @tyler-components-web/divider

To install the full TCW component library package, see the getting started guide.

Usage

Although this is the simplest of components in the TCW library, it removes the need for developers to worry about styles concerning dividing lines between sections.
