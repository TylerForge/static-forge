Checkbox

The checkbox component is a wrapper around the MDC checkbox.

Installation

This component can be installed as an individual npm package via @tyler-components-web/checkbox

To install the full TCW component library package, see the getting started guide.

Usage

It is intended that you get/set the value of the checkbox component through the exposed input element just as you would any other checkbox.

The tcw-checkbox component is responsible for binding to the input for  events and styling concerns only, but otherwise, the checkbox function just as a native checkbox element does.

Basic
<tcw-checkbox>
  <input type="checkbox" />
tcw-checkbox>
With label
<div class="tyl-form-field">
  <tcw-checkbox>
    <input type="checkbox" id="input-checkbox-01" checked>
  tcw-checkbox>
  <label for="input-checkbox-01">Checkbox inputlabel>
div>

Note the use of the wrapper div that contains a class of tyl-form-field. This is required to ensure that the label is positioned properly.
