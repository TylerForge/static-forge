Page State

This is a generic component that can be used for displaying various full-page states. Examples of this are empty search results screens, error pages, landing pages… etc.

Installation

This component can be installed as an individual npm package via @tyler-components-web/page-state

To install the full TCW component library package, see the getting started guide.

Guidance

Use this component when you want to display a graphic, with a message and optional actions within a page (typically a full page).

This component is used as a structural component to provide consistency, while allowing for variance in the content that it contains.

Usage

Nothing but tumbleweeds here...
Even our best explorer couldn't find the page you're looking for. It might have been removed or you may have mistyped the URL.
<tcw-page-state>
  <img src="images/404-spot-hero.svg" slot="graphic" />
  <div slot="title">Nothing but tumbleweeds here...div>
  <div slot="message">Even our best explorer couldn't find the page you're looking for. It might have been removed or you may have mistyped the URL.div>
  <tcw-button type="raised" slot="action">
    <button>Go backbutton>
  tcw-button>
  <tcw-button type="outlined" slot="action">
    <button>Refreshbutton>
  tcw-button>
tcw-page-state>

Slots

Name

Values

graphic

The element to display as the graphic. Note: by default 100% height and width is set on the elements that are projected into this slot, but can be overridden if necessary.

title

The element containing the title text.

message

The element containing the message text.

action

The action button element location.

CSS custom properties

Name

Description

--tyl-page-state-graphic-height

Controls the height of the graphic in a desktop viewport. Default is 296px.

--tyl-page-state-mobile-graphic-height

Controls the height of the graphic in a mobile viewport. Default is auto.

--tyl-page-state-width

Controls the width of component in a desktop viewport. Default is 576px.

--tyl-page-state-mobile-width

Controls the width of component in a mobileviewport. Default is 70%.

--tyl-page-state-vertical-margin

Controls the vertical margin spacing between the elements. Default is 56px.
