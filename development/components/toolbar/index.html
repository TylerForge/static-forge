Toolbar

The toolbar spans 100% width, uses a border to separate the toolbar from the content area, and has slots for placing any content at the start, center, and end. It can be used below the omnibar to display a page title and actions, within a nav drawer as a section, as a footer, or as header & footer in dialogs.

Installation

This component can be installed as an individual npm package via @tyler-components-web/toolbar

To install the full TCW component library package, see the getting started guide.

Slots

Name

Values

start

Places content at the far left of the toolbar.

center

Places content at the center of the toolbar.

end

Places content at the far right of the toolbar.

icon

Places content to the left of the label and value.

Properties/Attributes

Property

Attribute

Description

inverted

inverted

Controls whether a border bottom (default) or top (true) is used.

CSS custom properties

Name

Description

--tyl-toolbar-height

Controls the height.

--tyl-toolbar-padding

Controls the left and right padding.

--tyl-toolbar-border-bottom

Controls the style for the bottom border. Can be set to "none" for no border.

--tyl-toolbar-border-top

Controls the style for the top border. Can be set to "none" for no border.

--tyl-toolbar-border-top-left-radius

Sets the top left border radius.

--tyl-toolbar-border-top-right-radius

Sets the top right border radius.

--tyl-toolbar-border-bottom-left-radius

Sets the bottom left border radius.

--tyl-toolbar-border-bottom-right-radius

Sets the bottom right border radius.
