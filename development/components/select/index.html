Select

The select component provides a dropdown for a user to select from a list of predefined options.

Installation

This component can be installed as an individual npm package via @tyler-components-web/select

To install the full TCW component library package, see the getting started guide.

Usage

The select component options can be configured via HTML or JavaScript. When setting up the options via HTML, you can use the tcw-option element to configure the value and label for each option.

Basic
Bread, Cereal, Rice, and PastaVegetablesFruit
<tcw-select value="fruit" label="Food Group">
  <tcw-option value="">tcw-option>
  <tcw-option value="grains">Bread, Cereal, Rice, and Pastatcw-option>
  <tcw-option value="vegetables">Vegetablestcw-option>
  <tcw-option value="fruit">Fruittcw-option>
tcw-select>
Roomy

Use the roomy variant on mobile to allow for a larger touch target.

Bread, Cereal, Rice, and PastaVegetablesFruit
<tcw-select label="Food Group" density="roomy">
  <tcw-option value="">tcw-option>
  <tcw-option value="grains">Bread, Cereal, Rice, and Pastatcw-option>
  <tcw-option value="vegetables">Vegetablestcw-option>
  <tcw-option value="fruit">Fruittcw-option>
tcw-select>
Dense

Only use the dense variant sparingly.

Note: Do not use a label with the dense variant because the height of the element is too short. Place labels above or next to the element.

Bread, Cereal, Rice, and PastaVegetablesFruit
<tcw-select value="grains" density="dense">
  <tcw-option value="">tcw-option>
  <tcw-option value="grains">Bread, Cereal, Rice, and Pastatcw-option>
  <tcw-option value="vegetables">Vegetablestcw-option>
  <tcw-option value="fruit">Fruittcw-option>
tcw-select>
Always float

The component can be configured to keep the label floating at all times by setting the float-label-type attribute (or floatLabelType property in JavaScript) to always.

  <tcw-option value="">tcw-option>
  <tcw-option value="grains">Bread, Cereal, Rice, and Pastatcw-option>
  <tcw-option value="vegetables">Vegetablestcw-option>
  <tcw-option value="fruit">Fruittcw-option>
tcw-select>

Properties/Attributes

Property

Attribute

Description

value

value

The value of the selected.

selectedIndex

The selected index of the current option.

options

Gets/Sets the options programatically.

label

label

Gets/Sets the select label text.

multiple

multiple

Toggles the multiple select state.

open

Gets the open status.

disabled

disabled

Gets/sets the disabled state.

invalid

invalid

Controls the invalid state. When set to true, the label and border will change to the error theme color.

required

required

Controls the presence of an asterisk after the label to signify that the field is required.

density

Controls the density type. Valid values: default, roomy, dense. Use roomy for mobile, and don't use a label element with the dense type.

floatLabelType

float-label-type

The type of floating label to use. Valid values: auto (default), always.

popupElement

Gets the popup element when open.

placeholder

placeholder

Gets/sets the placeholder text for when no values are selected. This only applies when no label is provided.

Methods

Name

Values

setFocus()

Sets focus to the internal button element.

Slots

Name

Description

leading

Places content such as icons before the text.

helper-text

Places content below the component with the proper typography.

Events

Name

Description

change

Emits when the user chooses a different value. This is a custom event that contains the value and selectedIndex within the detail property of the event.

To get the value from the event, use event.target.value.

CSS Variables

Name

Description

--tyl-select-margin-top

Controls the margin top spacing above the host element.

--tyl-select-margin-bottom

Controls the margin bottom spacing below the host element.

--tyl-select-margin-bottom

Controls the margin bottom spacing below the host element.

--tyl-select-border-display

Controls the outline border display value. Default is flex.
