Open Icon

The open icon component is a simple component that encapsulates the functionality of an icon that spins based on the status of whether the open property is true or false.

Installation

This component can be installed as an individual npm package via @tyler-components-web/open-icon

To install the full TCW component library package, see the getting started guide.

Guidance

This component, as simple as it is, is best used in tandem with other components such as the expansion-panel component. It is encapsulates the functionality of hiding/showing/animating an icon from one position to another based on the status of a certain state.

Usage

Basic

Use the switch to toggle the open state.

<tcw-open-icon>tcw-open-icon>
Horizontal
<tcw-open-icon orientation="horizontal">tcw-open-icon>

Content for panel one.
