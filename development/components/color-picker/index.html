Color Picker

This component offers a UI for the user to select and adjust color values. It can produce values in various formats such as Hex, RGB and HSV.

Installation

This component can be installed as an individual npm package via @tyler-components-web/color-picker

To install the full TCW component library package, see the getting started guide.

Usage

This is a standalone component. To show this component in a popup, see the popup documentation.

<tcw-color-picker>tcw-color-picker>

Properties/Attributes

Property

Attribute

Description

value

value

Gets/sets the color value. Format must be in hex.

allowOpacity

allow-opacity

Controls the visibility of the opacity control.

rgba

Gets the current color in RGB(A) format.

hsva

Gets the current color in HSV(A) format.

opacity

Gets/sets the opacity.
