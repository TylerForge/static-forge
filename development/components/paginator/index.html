Paginator

The paginator component is used for splitting up content into smaller blocks that can be navigated by the user. The paginator is typically used to provide navigation through large data sets and/or search results, but can be used to provide paging functionality for any content.

Installation

This component can be installed as an individual npm package via @tyler-components-web/paginator

To install the full TCW component library package, see the getting started guide.

Usage

Default
<tcw-paginator total="100">tcw-paginator>

Try combining the paginator with the table component to allow for paging of data.

Properties/Attributes

Property

Attributes

Description

pageIndex

page-index

Gets/sets the current page index.

pageSize

page-size

Gets/sets the page size.

total

total

Gets/sets the total number of items to page through.

pageSizeOptions

Gets/sets the page size options for the select dropdown.

label

label

Gets/sets the label. Default is "Rows per page:".

firstLast

first-last

Controls the visibility of the first/last page buttons. Default is false.

disabled

disabled

Gets/sets the disabled state of the component.

Events

Name

Values

change

Emits when the current page has changed. The event.detail property contains the event data, and is of type IPaginatorChangeEvent.

Types

The following types are TypeScript interfaces that represent the expected data types that this component uses.

IPaginatorChangeEvent
export interface IPaginatorChangeEvent {
  type: string;
  pageSize: number;
  pageIndex: number;
}
