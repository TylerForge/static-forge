Date Picker

The datepicker component is used for allowing the user to enter a date through a text input, or by choosing a date from the popup calendar.

Installation

This component can be installed as an individual npm package via @tyler-components-web/datepicker

To install the full TCW component library package, see the getting started guide.

Guidance

The datepicker uses the tcw-popup internally to position the floating calendar.

It is expected that developers will provide the tcw-text-field to use for styling the input element of the datepicker.

Usage

Example
<tcw-datepicker>
  <tcw-text-field>
    <input type="text" id="input-datepicker">
    <label for="input-datepicker">Choose datelabel>
  tcw-text-field>
tcw-datepicker>

Properties

Name

Description

min

Sets the minimum date value.

max

Sets the maximum date value.

Events

Name

Description

input

Emits when the user is typing in the input.

change

Emits when the value changes by selecting a value in the dropdown.
