Omnibar

The Omnibar component is intended to be used as an application header, but it is flexible enough to support different variants, styles, colors… etc. It can also be used elsewhere within an application, such as within a dialog or map.

Going forward, this component is intended to be used in place of the legacy Omnibar to allow for flexibility in the content it displays, while still giving structure and alignment.

For in-depth implementation details regarding integrating an application with omnibar and any of the omnibar sub-components (menu button, search input, help button, app-launcher, profile button… etc.) please see the following guide:

Omnibar implementation guide

For API details regarding any of the omnibar sub-components please see one of the following pages:

  • App launcher
  • Omnibar search
  • Ominbar profile
  • Omnibar help button
  • Omnibar notification button
  • Action search (coming soon)

Installation

This component can be installed as an individual npm package via @tyler-components-web/omni

To install the full TCW component library package, see the getting started guide.

Guidance

The following examples show some common use cases for the Omnibar.

Default
<tcw-omnibar title-text="Basic Omnibar">tcw-omnibar>
Menu
<tcw-omnibar title-text="Omnibar">
  <tcw-omnibar-menu-button slot="menu">tcw-omnibar-menu-button>
tcw-omnibar>
Search
<tcw-omnibar title-text="Search">
  <tcw-omnibar-search slot="center">
      <input type="text" placeholder="Search" />
  tcw-omnibar-search>
tcw-omnibar>
Full
<tcw-omnibar title-text="Full">
  <tcw-omnibar-menu-button slot="menu">tcw-omnibar-menu-button>
  <tcw-omnibar-search slot="center">
      <input type="text" placeholder="Search" />
  tcw-omnibar-search>
  <tcw-omnibar-help-button slot="action">tcw-omnibar-help-button>
  <tcw-omnibar-notification-button slot="action">tcw-omnibar-notification-button>
  <tcw-omnibar-app-launcher-button slot="action" allow-more="true">tcw-omnibar-app-launcher-button>
  <tcw-omnibar-profile-button slot="profile" avatar-text="First Last" full-name="First Last" email="first.last@tylertech.com">tcw-omnibar-profile-button>
tcw-omnibar>

Slots

Name

Values

menu

Places content at the far left of the app bar. Generally a menu button is used here.

logo

Places content in the logo slot to the left of the title text. Use this slot to override the default Tyler logo.

center

Places content at the center of the app bar.

user-action

Places content between the center slot and the action slot. This is where user-specific actions should live.

action

Places content in the right-justified slot before the profile content. This is where global actions should live.

profile

Places content to the far right of the omnibar where the users profile information should live.

bottom

Adds a second row to the app bar. This allows for placing content such as tabs for example.

Properties/Attributes

Property

Attribute

Description

titleText

title-text

Gets/sets the title text.

prominent

prominent

Toggles the prominent variant. Default is false.

fixed

fixed

Toggles the fixed variant. Default is false.

raised

raised

Toggles the elevation state. Default is true.

scrollTarget

scroll-target

Sets the selector to use for locating the scroll container to listen to when using the "fixed" variant. This controls whether the elevation is shown when scrolled.

logo

logo

Controls the visibility of the default Tyler logo.

theme

Controls the theme color applied to the omnibar background.

Name

Values
