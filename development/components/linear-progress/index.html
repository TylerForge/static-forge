Linear Progress

The linear progress indicator can be used to display the length of a process, or animate indefinitely to express an unspecified wait time.

Installation

This component can be installed as an individual npm package via @tyler-components-web/linear-progress

To install the full TCW component library package, see the getting started guide.

Usage

Indeterminate
<tcw-linear-progress>tcw-linear-progress>
Determinate
<tcw-linear-progress determinate="true" progress="0.5" buffer="1">tcw-linear-progress>
Buffer
<tcw-linear-progress determinate="true" progress="0.33" buffer="0.66">tcw-linear-progress>
Custom color
<tcw-linear-progress style="--tyl-linear-progress-color: var(--mdc-theme-secondary); --tyl-linear-progress-buffer-color: lightgray;">tcw-linear-progress>

To change the color of the bar, set the --tyl-linear-progress-color CSS custom property to a color value of your choice.

Properties/Attributes

Property

Attribute

Description

determinate

determinate

The determinate state of the indicator. Default is false.

progress

progress

The progress amount (between 0 and 1) of the indicator. Only valid in determinate state. Default is 0.

buffer

buffer

The buffer amount (between 0 and 1) of the indicator. Only valid in determinate state. Default is 0.

Methods

Name

Description

open()

Shows the component if it is in the closed state.

close()

Closed the component if it is in the open state.

CSS custom properties

Name

Values

--tyl-linear-progress-color

Controls the color of the progress bar.

--tyl-linear-progress-buffer-color

Controls the color of the buffer bar.
