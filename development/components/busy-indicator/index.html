Busy Indicator

This component overlays a backdrop and centered message within a dialog-style component that offers configurable options for displaying a message, title, progress indicators… etc.

Installation

This component can be installed as an individual npm package via @tyler-components-web/busy-indicator

To install the full TCW component library package, see the getting started guide.

Usage

The use of JavaScript to show/hide the busy-indicator component is typically the way that it will be used. Below is an example of dynamically creating a busy-indicator instance and configuring it.

const busyIndicator = document.createElement('tcw-busy-indicator');
busyIndicator.titleText = titleTextbox.value;
busyIndicator.message = messageTextbox.value;

document.body.appendChild(busyIndicator);

Angular usage

The @tylertech/tyler-components-web-angular adapter library provides an easy-to-use service that can be injected into any component to show a busy indicator:

import { Component, ViewChild, ViewContainerRef } from '@angular/core';
import { BusyIndicatorService } from '@tylertech/tyler-components-web-angular';
import { BUSY_INDICATOR_CONSTANTS } from '@tylertech/tyler-components-web';

@Component({
  selector: 'app-busy-indicator',
  templateUrl: './busy-indicator.component.html',
  styleUrls: ['./busy-indicator.component.scss']
})
export class BusyIndicatorExampleComponent {
  constructor(private _busyIndicatorService: BusyIndicatorService) {}

  public showBusyIndicator(): void {
    const busyIndicator = this._busyIndicatorService.show({
      titleText: 'Title',
      message: 'Message...',
      cancel: true,
      spinner: true,
      progressBar: true,
      progressBarDeterminate: true,
      progress: 0.5,
      buffer: 0.75
    });

    busyIndicator.addEventListener(BUSY_INDICATOR_CONSTANTS.events.CANCEL, () => {
      console.log('Busy indicator cancelled.');
    });
  }

  public hideBusyIndicator(): void {
    this._busyIndicatorService.hide();
  }
}

Properties/Attributes

Property

Attribute

Description

title

title-text

Gets/sets the title text.

message

message

Gets/sets the message text.

cancel

cancel

Controls the visibility of the cancel button. Default is false.

spinner

Controls the visibility of the progress spinner.

progressBar

Controls the visibility of the progress bar. Default is false.

progressBarDeterminate

Controls whether the progress bar is determinate or not.

progress

Gets/sets the progress percentage of the progress bar.

buffer

Gets/sets the buffer percentage of the progress bar.

width

Sets the width of the surface element. Default is "auto".

isDestroyed

Gets the destroyed status of the component.

autoEllipsis

auto-ellipsis

Controls whether the message is appended with "..." when multiple messages are provided.. Default is true.

delay

Controls the visibility timeout when appended to the DOM. Default is 500.

Methods

Name

Description
