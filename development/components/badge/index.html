Badge

The badge component is intended to be used for displaying a status indicator that is related to another UI element. This typically contains numbers or short text descriptions.

Installation

This component can be installed as an individual npm package via @tyler-components-web/badge

To install the full TCW component library package, see the getting started guide.

Usage

The badge offers default styles, but virtually anything you will need to customize is available through the CSS custom properties (see below for details). You can adjust the badge to fit your needs, as well as use some of the built-in classes for positioning the badge on an tcw-icon-button for example.

Note the use of the tyl-icon-button--with-badge utility class that can be used to easily position the badge in the proper place of an icon-button. This also uses the positioned attribute on the tcw-badge component to easily set the position to absolute.

You can control the position of the badge manually by using the  top, right, bottom, left styles.

Properties/Attributes

Property

Attribute

Description

dot

dot

Sets the badge to a smaller height/width when no text content is to be displayed. Default is false.

open

open

Controls the visibility of the badge, while leaving it in the DOM. This applys a scale transition to hide/show the component.

theme

Allows for easily setting the theme color to use. Valid values: primary, secondary, tertiary.

positioned

Sets the position of the host element to absolute to allow for custom positioning.
