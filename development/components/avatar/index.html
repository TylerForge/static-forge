Avatar

This component will render an element with the letter(s) based on the value property/attribute as content. It has a configurable size, radius, and changes its background color based on the first letter that is displayed.

A URL can also be provided to display an avatar image.

Installation

This component can be installed as an individual npm package via @tyler-components-web/avatar

To install the full TCW component library package, see the getting started guide.

Usage

With text
<tcw-avatar text="First Last">tcw-avatar>
With image
<tcw-avatar image-url="https://some-url.com/image.png">tcw-avatar>
With icon and CSS custom property
face 
<tcw-avatar style="--tyl-avatar-background-color: #000000;">
  <i class="material-icons">facei>
tcw-avatar>

Properties/Attributes

Property

Attribute

Description

text

text

Gets/sets the text to display.

imageUrl

image-url

Sets the background image URL to use.

letterCount

letter-count

Controls the number of letters to display from the provided text. Default is 2.

autoColor

auto-color

Controls whether the background color set automatically based on the text value. Does not have any effect when an image URL is specified.

Slots

Name

Values

default

The default slot will override any content set via the text property/attribute.

CSS custom properties

Name

Values

--tyl-avatar-font-weight

Sets the font-weight of the text.

--tyl-avatar-background-color

Sets the background-color of the container.

--tyl-avatar-foreground-color

Sets the color of the text.

--tyl-avatar-size

Sets the height and width of the container.

--tyl-avatar-font-size

Sets the font-size of the text.

--tyl-avatar-radius

Sets the border-radius of the container element. Defaults to 50%.
