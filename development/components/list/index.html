List

The list component provides an opinionated style for vertical oriented items. The list items are flexible in content, while still providing structure/positioning and functionality to support various list types.

Installation

This component can be installed as an individual npm package via @tyler-components-web/list

To install the full TCW component library package, see the getting started guide.

Usage

The list component supports many different variants. Supported list types:

  • Single-line
  • Two-line
  • Three-line
  • Selectable
  • Checkbox
  • Radio
  • Static

A common use case is to use a list within the drawer component to create a side navigation drawer. See the Drawer example below for implementation details on this pattern.

List Properties/Attributes

Property

Attributes

Description

static

static

Sets the list to be static, meaning the list items are non-interactive.

selectedValue

selected-value

Sets the selected list-item value.

dense

dense

Controls whether all list-items are dense or not.

Note: When setting the static property on the element, the child elements will inherit the value.

List Item Properties/Attributes

Property

Attributes

Description

static

static

Sets the list item to be static, meaning it is non-interactive.

twoLine

two-line

Sets the height of this list item to accomodate two lines of text.

threeLine

three-line

Sets the height of this list item to accomodate three lines of text.

active

active

Controls the active state of the list item.

selected

selected

Controls the selected state of the list item.

value

value

Unique list item value. When selected, the list item will emit an event containing this value.

href

href

Gets/sets the URL that the browser will navigate to when clicked. This simulates an anchor tag.

target

target

Sets the window target when navigating to the provided href URL using the same API as an anchor tag. Only pertains when the href value is provided.

ripple

ripple

Controls the existence of the ripple effect.

dense

dense

Controls whether the list item is dense or not.

disabled

disabled

Controls the disabled state.

Description

Name

Description

Description

Description
