Card

The card component implements a material-styled card with default padding and margin. It allows for any content to be provided, while allowing slotted content specifically for header and footer to align content above/below the body content.

Installation

This component can be installed as an individual npm package via @tyler-components-web/card

To install the full TCW component library package, see the getting started guide.

Usage

Basic
Header
Body
Footer
<tcw-card>
  <div slot="header">Headerdiv>
  <div>Bodydiv>
  <div slot="footer">Footerdiv>
tcw-card>
Outlined

The outlined variant removes the raised box-shadow effect for a more flat design.

Outlined card
<tcw-card outlined="true">
  <div>Outlined carddiv>
tcw-card>

By default the card component will apply a padding of 16px to the entire container element. To disable the padding set the has-padding attribute to false.

Properties/Attributes

Property

Attribute

Description

hasPadding

has-padding

Controls whether the card has padding or not. Defaults to true.

outlined

outlined

Gives the card a light gray border and removes the drop shadow. Defaults to false.

Slots

Name

Description

header

Projects content aligned to the top of the card above the body.

default

Unamed slot for projecting content to the card body. This is where elements with no slot attribute will be projected.

footer

Projects content aligned to the bottom of the card below the body.

CSS custom properties

Name

Description

--tyl-card-padding

Controls the padding of the container element.

--tyl-card-header-padding

Controls the padding of the internal header element.

--tyl-card-body-padding

Controls the padding of the internal body element.

--tyl-card-footer-padding

Controls the padding of the internal footer element.

--tyl-card-body-overflow

Controls the overflow style of the internal body element.

--tyl-card-height

Sets the height of the card.

--tyl-card-width

Sets the width of the card.
