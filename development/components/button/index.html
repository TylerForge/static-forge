Button

The button component is simply a wrapper around the standard HTML 

Installation

This component can be installed as an individual npm package via @tyler-components-web/button

To install the full TCW component library package, see the getting started guide.

Usage

Below are common variants that the button component supports. The button element is left available to the developer to easily allow for binding and adjusting properties as needed.

Default (flat)
<tcw-button>
  <button type="button">Default buttonbutton>
tcw-button>
Raised
<tcw-button type="raised">
  <button type="button">Raised buttonbutton>
tcw-button>
Unelevated
<tcw-button type="unelevated">
  <button type="button">Unelevated buttonbutton>
tcw-button>
Outlined
<tcw-button type="outlined">
  <button type="button">Outlined buttonbutton>
tcw-button>
Leading Icon
<tcw-button type="outlined">
  <button type="button">
    <i class="material-icons" aria-hidden="true">favoritei>
    <span>Leading icon buttonspan>
  button>
tcw-button>
Trailing Icon
<tcw-button type="outlined">
  <button type="button">
    <span>Trailing icon buttonspan>
    <i class="material-icons" aria-hidden="true">favoritei>
  button>
tcw-button>
Dense

The dense variant uses the type attribute by appending on a -dense suffix after the desired type name.

<tcw-button type="outlined-dense">
  <button type="button">Dense buttonbutton>
tcw-button>
CSS Only (low-fidelity)

All of the button classes are exported in the global stylesheet (or available in an individual stylesheet and sass mixins if desired). This will allow you to get the same look-and-feel for a low fidelity button, without using the web component if desired.

The component is responsible for adding the ripple effect, as well as making it easy to change the type and not worry about specific CSS class names.

<button type="button" class="tyl-button tyl-button--raised">CSS Only Buttonbutton>
