Tooltips

The tooltip component is a generic component that allows for placing a tooltip that dynamically shows/hides based on mouse interaction with its target element. This component is intended to be used for displaying a short text label next to an element when the mouse either hovers or longpresses on the element.

Installation

This component can be installed as an individual npm package via @tyler-components-web/tooltip

To install the full TCW component library package, see the getting started guide.

Guidance

Use tooltips to help provide guidance about what the meaning of a certain UI element is. On mobile, the tooltips can be displayed by long-pressing on the element.

The tooltip component is set up for accessibility by default.

Default
Some useful tooltip text
<tcw-button>
  <button>HOVER ME<button>
  <tcw-tooltip>Some useful tooltip texttcw-tooltip>
tcw-button>

The tooltip will use smart positioning to dynamically change its position around the target element if it attempts to render off screen.

Properties/Attributes

Property

Attributes

Description

text

text

The text to display in the tooltip.

builder

A builder function that returns a string or HTML element to render inside the tooltip dynamically.

target

target

A CSS selector to use for finding the target element.

delay

delay

A delay value (in milliseconds) to wait before display the tooltip when interacting with the target element.

position

position

The tooltip position around the target element. Valid values: top, right (default), bottom, left, top-left, top-right, bottom-left, bottom-right, left-top, left-bottom, right-top, right-bottom.
