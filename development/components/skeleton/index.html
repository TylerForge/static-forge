Skeleton

The skeleton component is a generic, low-level component that can be used to build loading placeholder pages/elements called “skeleton screens”. These screens represent the block-level visual elements of a page, or section of a page, where content is intended to be loaded sometime in the future. Using skeleton screens gives the illusion of speed to the user, and can often reduce frustration for users during long running processes.

Installation

This component can be installed as an individual npm package via @tyler-components-web/skeleton

To install the full TCW component library package, see the getting started guide.

Guidance

The skeleton component is a low-level component, that can be styled using regular CSS to create the specific placeholder element design you need.

With that being said, there are a few built-in variants that can be used by adding an attribute to the element. These built-in variants, are specifically designed to mimic other components in the TCW library for ease of use.

This component will produce a grey colored element, with a subtle animation over the to show that the content is loading.

Attributes

Name

Description

avatar

Mimics the avatar component

list-item

Matches the styles of an individual list-item.

text

Intended to give the impression that there is textual content in this location.

stretch

Intended to mimic larger sections of content where the skeleton will stretch to fit its parent bounds.

chip

Represents an individual chip-styled placeholder.

button

Represents an individual button-styled placeholder.

form-field

Represents a form field such as text-field, select, or autocomplete... etc.

CSS custom properties

Name

Description

--tyl-skeleton-height

Controls the internal height of the skeleton container.

--tyl-skeleton-margin

Controls the margin around the internal skeleton container element.

--tyl-skeleton-border-radius

Controls the border radius of the element. Defaults to 4px.

--tyl-skeleton-background-color

Controls the background-color of the internal skeleton element.
