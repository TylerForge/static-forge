Drawer

The drawer component is used to organize access to destinations and other functionality within an application. It can be positioned permanently next to content on the page, or can be dismissible and float over other content.

Installation

This component can be installed as an individual npm package via @tyler-components-web/drawer

To install the full TCW component library package, see the getting started guide.

Usage

The tcw-drawer component facilitates the open/close animation of content on either the left or right side of the screen. It does not restrict you to any specific content, but is most commonly used together with a tcw-list to create a navigation drawer, as seen below.

<tcw-drawer>
  <tcw-list>
    <tcw-list-item>Item Onetcw-list-item>
    <tcw-list-item>Item Twotcw-list-item>
    <tcw-list-item>Item Threetcw-list-item>
  tcw-list>
tcw-drawer>

Properties/Attributes

Property

Attribute

Description

open

open

Toggles the open state of the drawer when dismissible or modal type.

direction

direction

Controls the animation direction of the drawer for positioning on the left vs. right side of the screen. Default is left.

type

type

Sets the type of drawer. Valid values: "permanent" (default), "dismissible", "modal", "mini" and "mini-hover".

Events

Name

Values

tcw-drawer-close

Emits when the backdrop is clicked only when the modal type is used.

Slots

Name

Description

The default slot will place content within the scrollable drawer body.

header

Places content fixed at the top of the drawer.

footer

Places content fixed at the bottom of the drawer.
