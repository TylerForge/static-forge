Toast

The toast component is used for displaying a short non-intrusive notification to the user with an optional action button.

Installation

This component can be installed as an individual npm package via @tyler-components-web/toast

To install the full TCW component library package, see the getting started guide.

Usage

This component will typically be created on the fly in JavaScript, and it will automatically be disposed and removed from the DOM when its duration has been reached.

However, the toast element can be created using static HTML if desired:

<tcw-toast message="Save successful.">tcw-toast>

Below is an example of creating a toast via JavaScript:

const toast = document.createElement('tcw-toast');
toast.message = 'Save successful.';
document.body.appendChild(toast);
Basic example

Angular Usage

Usage within Angular is slightly different, and can be made easier by installing the @tylertech/tyler-components-web-angular adapter package.

After doing so, there is an Angular provider available called ToastService that can be injected and used to show a toast on demand:

import { ToastService } from '@tylertech/tyler-components-web-angular';
import { PopupPlacement } from '@tylertech/tyler-components-web';

@Component({
  selector: 'app-toast-example',
  templateUrl: './toast-example.component.html'
})
export class ToastExampleComponent {  
  constructor(private _toastService: ToastService) {}

  public onShowToast(): void {
    const toast = this._toastService.show({
      message: 'Save successful.',
      actionText: 'UNDO',
      actionHandler: () => toast.hide(),
      placement: PopupPlacement.Bottom,
      duration: 2000
    });
  }
}

Properties/Attributes

Name

Attribute

Description

message

message

The message text.

actionText

action-text

The action button text. Controls the visibility of the action button as well.

duration

duration

The length of time in milliseconds to show the toast.

placement

placement

Sets the placement of the toast around the screen. Valid values: bottom (default), bottom-left, bottom-right, top-left, top, top-right.

showClose

show-close

Controls the visibility of the close button. Default is true.

builder

n/a

Sets the builder function that will be called in place of the message text to allow for providing a custom HTML template.

Methods

Name

Description

hide()

Hides the toast and removes it from the DOM.

Events

Name

Description

tcw-toast-action

Emits when the action button is clicked.

tcw-toast-close

Emits when close button is clicked.
