Scaffold

Use the scaffold component to easily compose common layouts such as full applications layouts with header, scrollable body, sidenav, and footer. Create layouts within dialogs or cards that require the use of a header, scrollable content area, and footer. You can nest the scaffold component as deeply as you’d like to obtain complex layouts with little to no CSS required.

Installation

This component can be installed as an individual npm package via @tyler-components-web/scaffold

To install the full TCW component library package, see the getting started guide.

Guidance

When using this component, it will be sized to match the viewport height and width by default. This can be overridden to set a specific height and/or width if desired using the CSS custom properties noted below. An example of doing something like this would be if you want to nest tcw-scaffold components within each other, or use it in a dialog.

Note: When nesting tcw-scaffold components, it may be necessary to adjust any direct parent elements to match the height and width of the body container. This will allow the scaffold to stretch to match the height and width of its parent when using the CSS custom properties noted below.

Properties/Attributes

Property

Attribute

Description

stretch

Sets the --tyl-scaffold-height and --tyl-scaffold-width CSS custom properties to a value of 100%.

Slots

Name

Description

header

This slot places content at the top of the viewport.

body

This slot places content below the header slot and expands to the bottom of the viewport, or to the footer if specified.

body-header

This slot places content above the scrollable body slot.

body-left

This slot places content to the left of the body.

body-right

This slot places content to the right of the body.

body-footer

This slot places content at the bottom of the body.

footer

This slot places content below the body at the bottom of the viewport.

CSS custom properties

Name

Description

--tyl-scaffold-height

Controls the height of the root element. Default value is 100vh.

--tyl-scaffold-width

Controls the width of the root element. Default value is 100vw.
