Floating Action Button

Floating action buttons represents the primary action in an application. Only one floating action button is recommended per screen to represent the most common action.

Installation

This component can be installed as an individual npm package via @tyler-components-web/floating-action-button

To install the full TCW component library package, see the getting started guide.

Guidance

FABs are static by default, meaning they will be rendered in the layout where they are placed. It is up to the developer to position the FAB within their own application’s design.

Create a class that can be applied to your component to position it absolutely:

.app-fab--absolute {
  position: fixed;
  bottom: 1rem;
  right: 1rem;
}

Custom color
<tcw-fab style="--mdc-theme-text-primary-on-light: white; --mdc-theme-secondary: #3f51b5;">
  <button type="button" aria-label="Create">
    <i class="material-icons" aria-hidden="true">addi>
  button>
tcw-fab>

Note the use of the --mdc-theme-text-primary-on-light CSS custom property. This is required to ensure that the proper contrast ratio is achieved for accessibility. The default black font color on the primary color background would fail an accessibility audit, making this component hard to use for color blind users.

Properties/Attributes

Property

Attribute

Description

color

color

The theme type to apply.

mini

mini

Displays the FAB in its mini state.

extended

extended

Allows for a label to be used.

exited

exited

Controls the exited state. This animates the FAB in/out while leaving it in the DOM.

Description
