Chips

Chips are compact elements that allow users to enter information, select a choice, filter content, or trigger an action.

Installation

This component can be installed as an individual npm package via @tyler-components-web/chips

To install the full TCW component library package, see the getting started guide.

Usage

Chips can have one of the following types:

  • Choice – Allows for selecting only one chip at a time
  • Filter – Allows for selecting multiple chips.
  • Action – A chip that acts as button for performing an action.
  • Input – A chip that is “deletable” and can be used to transform text into a chip element.
Basic
<tcw-chip text="Label">tcw-chip>
Choice chips
<tcw-chip-set type="choice">
  <tcw-chip text="Small">tcw-chip>
  <tcw-chip text="Medium" selected="true">tcw-chip>
  <tcw-chip text="Large">tcw-chip>
tcw-chip-set>
Filter chips
<tcw-chip-set type="filter">
  <tcw-chip text="Tops" selected="true">tcw-chip>
  <tcw-chip text="Bottoms" selected="true">tcw-chip>
  <tcw-chip text="Shoes">tcw-chip>
  <tcw-chip text="Accessories">tcw-chip>
tcw-chip-set>
Action chips
<tcw-chip-set type="action">
  <tcw-chip icon="event" text="Add to Calendar" value="calendar">tcw-chip>
  <tcw-chip icon="bookmark" text="Bookmark" value="bookmark">tcw-chip>
  <tcw-chip icon="alarm" text="Set Alarm" value="alarm">tcw-chip>
  <tcw-chip icon="directions" text="Get Directions" value="directions">tcw-chip>
tcw-chip-set>
Input chips
<tcw-chip-set type="input">
  <tcw-chip icon="place" text="Falmouth" delete="true">tcw-chip>
  <tcw-chip icon="place" text="Yarmouth" delete="true">tcw-chip>
  <tcw-chip icon="place" text="Plano" delete="true">tcw-chip>
  <tcw-chip icon="place" text="Renton" delete="true">tcw-chip>
tcw-chip-set>

Tip: By setting the type on the tcw-chip-set element, the tcw-chip elements can omit it, and they will inherit the type from the chip set when they are instantiated. Each chip can set its own type if desired though.

Chip Properties

Property

Attributes

Description

type

type

Sets the type of chip.

icon

icon

Gets/sets the chip icon.

text

text

Gets/sets the chip text.

delete

delete

Gets/sets whether the delete button is visible for this chip. Valid only for input chips.

selected

selected

Gets/sets the selected state. Valid only for choice and filter chips.

value

value

Gets/sets the value of this chip.

Chip Set Properties

Property

Attributes

Description

orientation

orientation

Sets the orientation of the chip set. Default: horizontal.

type

type

Sets the type of chips that this chip set contains.

Events

Name

Description

tcw-chip-select

Emits when an action, filter, or choice chip is clicked.

tcw-chip-delete

Emits when an input chips' delete button is clicked.

Note: These event bubble and can be listened for from either the tcw-chip or tcw-chip-set elements.

interface IChipSelectEventData {
