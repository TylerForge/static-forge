Expansion panel

This component encapsulates the functionality of expanding/collapsing content.

Installation

This component can be installed as an individual npm package via @tyler-components-web/expansion-panel

To install the full TCW component library package, see the getting started guide.

Usage

This component does not provide any styling for the content that is displayed within it. It simply handles click events on the slotted header element to expand/collapse the content element.

The header element is technically optional if you would only like to manage the open state programmatically.

Properties/Attributes

Property

Attribute

Description

open

open

Controls whether the content is expanded (true) or collapsed (false). Default is false.

orientation

orientation

Controls the expand/collapse animation direction. Valid values: "vertical" (default), "horizontal".

Note: when using the "horizontal" orientation, set a width for the default slot to ensure a smooth transition animation.

Slots

Name

Description

header

Projects content into the always visible area above the expandable content. This is optional

default

The default unnamed slot contains the expandable content.

Events

Name

Description

tcw-expansion-panel-open

Emits when the panel is opened.

tcw-expansion-panel-close

Emits when the panel is closed.
