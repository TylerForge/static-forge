Accordion

The accordion component is a wrapper component that is intended to contain tcw-expansion-panel  components as children. 

This component acts as an orchestrator of the expansion panel child components to ensure that only one panel is expanded at any given time. It will close the open panel when another one is expanded.

The expand/collapse functionality is within each tcw-expansion-panel child element.

Installation

This component can be installed as an individual npm package via @tyler-components-web/accordion

To install the full TCW component library package, see the getting started guide.

Guidance

This component does not provide any visual appearance. It is just an orchestrator of child tcw-expansion-panel components. See the Expansion panel component documentation for information on how to use that component.

Properties/Attributes

Property

Attribute

Description

panelSelector

panel-selector

Gets/sets the selector to use for finding the child expansion panels. Defaults to searching the direct children for elements.

Accessibility

  • Verify that you can tab to each panel in the accordion.
    • Ensure that there is a visual cue that the panel is selected.
  • Verify that pressing the space bar or enter key while tab-focusing on a panel will toggle it open and closed.
  • If any open panel contains a link, button, or other navigation element, ensure that those tab stops are included while tabbing through the accordion.
  • If any closed panel contains a link, button, or other navigation element, ensure those tab stops are skipped while tabbing through the accordion.
