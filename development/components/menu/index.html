Menu

The menu component will display a list of options that a user can choose from within a dropdown-style popup. The menu will be triggered automatically by the child component that it wraps, such as a 

Installation

This component can be installed as an individual npm package via @tyler-components-web/menu

To install the full TCW component library package, see the getting started guide.

Guidance

The menu component is intended to be used as a wrapper around its “trigger” element, typically a button. Whenever the trigger element is clicked, the menu will be shown.

The use of JavaScript is currently required to configure the options that are displayed.

Usage

Basic
<tcw-menu>
  <tcw-button type="raised">
    <button type="button">Show menubutton>
  tcw-button>
tcw-menu>

Set the menu options via JavaScript:

menu.options = [
  { value: 'edit', label: 'Edit' },
  { value: 'delete', label: 'Delete' },
  { value: 'view', label: 'View' }
];

Properties/Attributes

Property

Attribute

Description

open

open

Controls the open state of the menu popup.

placement

placement

Gets/sets the popup placement around the child element.

selectedIndex

selected-index

Gets/sets the selected menu item index.

options

Sets the IMenuOption items to display. If an MenuOptionFactory is provided as the value instead, then that will be used to fetch the options asynchronously.

dense

dense

Controls dense state of the list options in the dropdown.

Events

Name

Values

tcw-menu-select

Emits when a menu item is selected from the dropdown. The data can be accessed via the event.detail property, and is of type IMenuSelectEventData.
