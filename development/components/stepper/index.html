Stepper

The stepper component displays progress through a sequence of logical and numbered steps.

Installation

This component can be installed as an individual npm package via @tyler-components-web/stepper

To install the full TCW component library package, see the getting started guide.

Usage

Basic
<tcw-stepper>tcw-stepper>

The following JavaScript is used to configure and receive events from the component:

const stepper = document.querySelector('tcw-stepper');
stepper.steps = [
  { label: 'Step one', completed: true },
  { label: 'Step two', optionalLabel: 'Optional', completed: true, editable: true },
  { label: 'Step three' },
  { label: 'Step four' },
  { label: 'Done' }
];
stepper.selectedIndex = 2;
stepper.addEventListener('tcw-stepper-select', (evt: CustomEvent) => {
  stepper.selectedIndex = evt.detail; // The detail property contains the selected index.
});

Note: This component does not allow for declarative markup at this time, but will in the future. For now, all interaction with the component is expected to be done through the use of JavaScript.
