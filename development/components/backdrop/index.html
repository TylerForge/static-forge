Backdrop

The backdrop component is intended to be used as an optional building block for other components (such as the dialog). It is used to display a semi-opaque area behind a floating element to convey focus to the floating element. This also stops interaction with the elements behind the floating elements.

Properties/Attributes

Property

Attribute

Description

delay

delay

Sets the delay (in milliseconds) before the fade-in animation occurs.

maxOpacity

max-opacity

Sets the maximum opacity that the element will fade-in to.

Methods

Name

Description

fadeOut()

Starts the fade out animation to 0% opacity.

fadeIn ()

Starts the fade in animation to max opacity.

Events

Name

Description

tcw-backdrop-click

Emitted when the backdrop receives a click event.
