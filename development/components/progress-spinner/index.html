Progress Spinner

The progress spinner component is an animating circular activity indicator.

Installation

This component can be installed as an individual npm package via @tyler-components-web/progress-spinner

To install the full TCW component library package, see the getting started guide.

Usage

This component renders as an svg, and the properties can be controlled dynamically through the component and CSS custom properties.

Note: On IE 11, the animation gracefully degrades to a lower-fidelity animation.

Default
<tcw-progress-spinner>tcw-progress-spinner>
Custom
<tcw-progress-spinner 
  style="--tyl-progress-spinner-color: var(--mdc-theme-secondary);" 
  stroke-width="5"
  diameter="50">
tcw-progress-spinner>

Properties/Attributes

Property

Attribute

Description

mode

mode

The mode indeterminate, determinate.

progress

progress

The progress amount between 0 and 100.

diameter

diameter

The diameter in pixels.

strokeWidth

stroke-width

The width of the stroke in pixels.

CSS custom properties

Name

Description
