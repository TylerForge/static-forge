Properties/Attributes

Property

Attribute

Description

density

density

Controls the density type. Valid values: default, roomy, dense. Use roomy for mobile, and don't use a label element with the dense type.

floatLabelType

float-label-type

The type of floating label to use. Valid values: auto (default), always.

shape

shape

The outline shape type to use. Valid values: default, rounded.

invalid

invalid

Controls the invalid state. When set to true, the label and border will change to the error theme color.

required

required

Controls the presence of an asterisk after the label to signify that the field is required.

Methods

Name

Description

floatLabel()

Controls the floating state of the label. Call this to manually float the label.
